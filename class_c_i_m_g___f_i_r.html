Подробное описание

template<class nmbits_in, class nmbits_out>
class CIMG_FIR< nmbits_in, nmbits_out >

Класс КИХ фильтра

Параметры шаблона
nmbits_inТип указывающий разрядность входного изображения. Допустимые типы : nm8s,nm16s,nm32s,nm64s
nmbits_outТип указывающий разрядность выходного изображения. Допустимые типы : nm8s,nm16s,nm32s,nm64s . Разрядность входного вектора не должна превышать разрядности выходного.

При вычислении первых и последних nKerHeight/2 сторк проивзодится выход за границы входного массива pSrcImg . Для коректного поведения функции необходимо дополнительные резервировать поля размером не менее nImgWidth(nKernHeight/2+1) нулевых элементов перед началом и в конце массива pSrcImg.
