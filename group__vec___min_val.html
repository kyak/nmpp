void vec_MinVal (nmreg nb1, nmreg f1cr, nmreg ar0, nmreg gr0, nmreg ar4, nmreg gr5, nmreg ar6)
 

Подробное описание

Поиск минимумов в колонках матрицы SrcMtr1.

Аргументы
nb1задает разбиение на колонки (необходимо wtw)
f1crзадает функцию активации
ar0указатель на столбец SrcMtr1
gr0SrcMtr stride
ar4указатель на временный массив размером nm64s[64]
gr5Высота матрицы SrcMtr1 N = [32,64,...]
Возвращаемые значения
ar6указатель на 64р. слово результатов (максимумов)
Restrictions:
При выходе из функции изменяется содержимое регистров: ar0,gr0,ar4,ar3,ar6,gr7.
